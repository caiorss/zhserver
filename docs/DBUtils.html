sqlDeleteRowsWhereID :: String -> String -> Int -> DBConn () Source #

Delete all rows from [table] where [column] = keyID.

Example:

  > sqlDeleteRowsWhereID "itemID" "items" 200

It executes the sql statement:

  > DELETE FROM items WHERE itemID = 200
